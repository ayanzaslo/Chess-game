# Chess-game
Project


Scurtă descriere:
Aplicație de tip consolă scrisă în limbajul C care permite jucarea unui meci de șah între doi jucători.
Respectă regulile oficiale ale jocului și validează mutările în timp real.
Proiectul pune accent pe logică, structuri de date și gestionarea stărilor jocului.

Specificații de bază:

Joc 1v1 (doi jucători pe același calculator)

Reprezentarea tablei de șah (8x8) în consolă

Mutări introduse de la tastatură (ex: e2 e4)

Validarea mutărilor conform regulilor fiecărei piese

Alternarea turelor între jucători

Detectarea stării de „șah”

Detectarea stării de „șah mat”

Mesaje de eroare pentru mutări invalide

Specificații suplimentare:

Implementarea regulilor speciale:

Rocadă (mică și mare)

Promovarea pionului

Istoric al mutărilor

Posibilitate de restart al jocului

Afișare îmbunătățită a tablei (cu coordonate)

Salvarea și încărcarea unei partide (opțional)

Modularizarea codului (fișiere separate pentru logică, input, afișare)
